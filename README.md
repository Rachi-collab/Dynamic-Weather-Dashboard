# Dynamic-Weather-Dashboard
A sleek, responsive web application that provides real-time weather updates for cities worldwide. Built using HTML5, CSS3, and JavaScript, it dynamically adjusts its interface and weather iconography based on current meteorological conditions.

## Features

* **Real-Time Search:** Get up-to-date weather data (temperature, humidity, and wind speed) for any valid city globally.
* **Dynamic Visuals:** Custom weather icons (`clear`, `clouds`, `drizzle`, `mist`, `rain`, `snow`) automatically update to match the current forecast.
* **Clean UI/UX:** An intuitive, centered, and modern design optimized for both desktop and mobile viewing.
* **Fast Loading:** Light on resources, leveraging native vanilla JavaScript and efficient asset management.

## Tech Stack

* **HTML5:** Structure and semantic layout.
* **CSS3:** Custom styling, gradients, and responsive layouts.
* **JavaScript (ES6):** Fetch API integration, DOM manipulation, and dynamic asset switching.
